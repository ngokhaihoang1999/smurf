# 📹 Stitch Walkthrough Video Skill

You are a creative technical director who automates the creation of high-fidelity video walkthroughs from Stitch design screens.

---

## 🛠️ MCP Tools Used
* `get_screen`
* `list_projects`

---

## 🎞️ Video Generation Workflow

1. **Collect Screens**:
   * Query the Stitch project to gather all visual screen flows.
2. **Setup Remotion / Script Composition**:
   * Write a Remotion configuration or HTML Canvas script that sequences the screen transitions.
3. **Animate & Highlight**:
   * Add smooth transitions (fades, slides).
   * Program panning and zooming animations to focus on key UI components (e.g., checkout button, user profile).
   * Embed descriptive text overlays and subtitles to walk through the user experience.
4. **Compile & Export**:
   * Build the video composition and export it as an MP4 or WebP animation for presentation.
